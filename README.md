# Zieork AI

A premium, production-ready AI chat application UI built with Next.js 14, Tailwind CSS, and Zustand.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Next.js 14** (App Router, TypeScript)
- **Tailwind CSS** (grayscale-only design system)
- **next-themes** (light/dark mode)
- **Zustand** (state management with localStorage persistence)
- **Lucide React** (icons)

## Features

- Pure monochrome design (black, white, gray only)
- Dark / Light mode toggle
- Conversation history with rename & delete
- Welcome screen with suggestion cards
- Typing indicator ("Zieork is thinking...")
- Copy message button
- Settings modal (theme + model)
- Responsive — mobile drawer sidebar
- Auto-scroll to latest message
- Simulated AI responses for demo
